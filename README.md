## 📖 About

A curated collection of coding problems organized by difficulty — from beginner-friendly exercises to advanced algorithmic challenges. This repo is built to help sharpen problem-solving skills progressively: starting with the fundamentals like loops, arrays, and basic logic, and working up toward dynamic programming, graphs, and real interview-style challenges.

Each problem includes a clear statement and, where relevant, notes on approach and time/space complexity.

## 🛠️ How to Use

1. Clone the repo
   ```bash
   git clone https://github.com/syedrasibali/problems-beginner-to-advanced.git
   ```
2. Pick a difficulty folder that matches where you are
3. Solve, compare, and iterate
4. Contributions and PRs are welcome!
